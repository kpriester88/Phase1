# Session 07 Reflection

## Why Lists Beat Variables at Scale

A list is a better container for 100 IP addresses than 100 seperate variables because it can be iterated with a loop.
The for loop will go through each string that is part of the array, rather than to type out x amount of variables. The computer can read and loop through each one faster than to manually type each one out.
The manual labor would be entirely too time consuming and mistakes can be made typing each one out as well.
